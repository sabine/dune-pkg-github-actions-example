# Dune Package Management GitHub Actions Example

This is a minimal hello-world OCaml repo that uses
[ocaml-dune/setup-dune](https://github.com/ocaml-dune/setup-dune) with Dune
Package Management.

Blog post: https://makerprism.com/en/blog/dune-package-management-on-github-actions/

## Build locally

```bash
dune build
```

## CI

See `.github/workflows/ci.yml` for the GitHub Actions setup.
